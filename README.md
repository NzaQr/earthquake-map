# Earthquake Map

Interactive map that displays earthquake data from the [USGS Earthquake API](https://earthquake.usgs.gov/fdsnws/event/1/).

## Requirements

- Node.js 18+
- npm

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
